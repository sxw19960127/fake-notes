# `vue-router`

```shell
// 下载
npm install vue-router --save
```

- 在`src`文件夹下新建一个`router`文件夹。
- 在`router`文件夹下新建一个`index.js`文件，用以配置相关的路由信息。

使用`vue-router`的步骤：

- 创建路由组件
- 配置路由映射：组件和路径映射关系
- 使用路由：通过`<router-link>`和`<router-view>`

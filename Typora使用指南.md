==Markdown== 常用语法介绍

**标题**：

```
# + space 一级标题
## + space 二级标题
### + space 三级标题
[TOC] 生成目录
```

**字体格式**：

```
**加粗**
*倾斜*
==高亮== 使用前,需在偏好设置中开启此功能
~~删除线~~
<u>下划线</u>
<center>居中</center>
```

**常规**：

```
ctrl + l 选中当前行
ctrl + d 选中当前单词
ctrl + f 搜索
ctrl + h 替换
```

**引用**：

```
[超链接](http://www.baidu.com)
![图片](http://img/iy.png)
> + space 文字引用 
```

**列表**：

```
- + space 无序列表
1. + space 有序列表
ctrl + T 插入表格
```

**表情**：

```
:happy:
:sad:
:cry:
```
